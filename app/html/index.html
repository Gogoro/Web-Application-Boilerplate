Web Application Boilerplate (WAB)

Boilerplate template to kickstart a web application. Including Sass, Grunt, LiveReload...

Includes:

  • FontAwesome - v.4.0.3
  • jQuery - v.1.11.1
  • Sass - v.?
  • Grunt - v.?
  • LiveReload - v.?

Initialization:

  1. Clone the repository into your desired location: git clone https://github.com/Gogoro/wab.git
  2. Change git remote path to your decired location git remote set-url urlhere.git
  3. Link your vHost to the app folder
  4. Install dependencies in your project root with the following shell command: npm install
  5. Run grunt with the following shell command: grunt
